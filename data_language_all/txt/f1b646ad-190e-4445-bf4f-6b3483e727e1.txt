libkvq
Library implementing key/value abstraction over multiple stores.
Dependencies

C compiler
CMake
conan

Developer workflow
mkdir cmake-build-debug && cd $_
conan install ..
cmake -DCMAKE_BUILD_TYPE=Debug -DTEST_ENABLED=1 ..
ctest

Run tests
Starting from directory created above, after make step, run:
./bin/test_*
