rkgttr-polyfills






Collection of polyfills I frequently use.

How to Install
sh
$ npm install rkgttr-polyfills --save-dev
or
sh
$ yarn add rkgttr-polyfills --dev
Getting Started
js
import 'rkgttr-polyfills';
or:
js
require('rkgttr-polyfills');
