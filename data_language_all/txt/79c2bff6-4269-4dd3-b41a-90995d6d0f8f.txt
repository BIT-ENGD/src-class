NODE-ENV-FILE 
Parse and load environment files (containing ENV variable exports) into Node.js environment, i.e. process.env.
Example
.env

.env2

index.js

API


(filepath)
javascript
env('./path/to/.env');


(filepath, options)
javascript
env('./path/to/.env', {verbose: true, overwrite: true});


Installation
shell
  $ npm install node-env-file
Test
Local tests:
shell
  $ make test
Examples
Local examples:
shell
  $ make example
Related Libraries

node-env-flag

Notes
Bash based approach:
Makefile
