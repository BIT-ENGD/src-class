BlogTracker-GTM
BlogTracker-GTM is a Javascript snippets for analyze blog behavior
Enanched with Google Tag Manager
CSS Class used for tracking:

Events:

Metrics:
timeToScroll ==> Time when user start scroll the page
timeToContentEnd ==> Time for reach end of article
totalTime ==> Total time of reading
Dimensions:
typeReader ==> Check if users is reeding in "Scanner" or "Reader" Mode
Installation and use

Install the script as a custom html tag and set as trigger all PageView
Add the needed dimensions and metrics to Analytics
For each event create a corresponding event on the GA taking care to pass the metrics and dimensions previously created
