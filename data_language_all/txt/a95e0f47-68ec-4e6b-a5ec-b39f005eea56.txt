Emacs.d
This repository contains my emacs configuration largely inspired from Prelude repositories.
It has been tested on Emacs 28+
Installation
Clone this repository in your home directory
git clone --recursive https://github.com/PierreRambaud/emacs.d ~/.emacs.d
With Cask
Install Cask by following theses instructions.
cd ~/.emacs.d && cask install
