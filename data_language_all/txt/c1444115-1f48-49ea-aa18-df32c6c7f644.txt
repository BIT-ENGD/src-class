使用方法
1、安装 wget 和 screen 命令
Centos:
yum install -y wget
yum install -y screen
Debian/Ubuntu:
apt-get install -y wget
apt-get install -y screen
2、然后运行安装命令

如果已经安装过，或安装出错后重新安装，请先 Uninstall AMH 4.2
更多模块可在管理面板进行安装。
