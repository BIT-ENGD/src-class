Linked-Data-Vocabulary-Editor
Collaborative Linked Data Vocabulary Editor and publishing platform based on Callimachus
Installation instructions:

Go to http://callimachusproject.org/ and install the callimachus following the up to date documentation.
Read the documentation: http://callimachusproject.org/documentation.xhtml?view ;)
Create folder(s) - for example /ns/src/ - and import the vocabulary-editor.car to the folder
Create new vocabulary from the Vocabulary class /ns/src/Vocabulary. Click "Create a new Vocabulary" from the main menu.
Edit the vocabulary and add Classes and Properties
