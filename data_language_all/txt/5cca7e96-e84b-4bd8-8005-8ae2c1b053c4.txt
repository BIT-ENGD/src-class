php-stacktrace
Read stacktrace from outside PHP process, inspired by ruby-stacktrace.
Install

Download or build php-stacktrace from source

Build from source
Clone and build this project:
git clone https://github.com/oraoto/php-stacktrace.git
cd php-stacktrace
cargo build
