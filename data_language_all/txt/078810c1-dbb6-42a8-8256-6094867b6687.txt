Installation


Install a full Ruby development environment


Install Jekyll and bundler (Ruby) gems:
bash
$ gem install jekyll bundler


Install the dependencies

Install the dependencies of Jekyll:
bash
$ bundle install # or
$ bundle update

Serve locally

Serve locally:
bash
$ bundle exec jekyll serve
