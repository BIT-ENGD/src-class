Requis:

Wamp
Git
Composer : https://getcomposer.org/Composer-Setup.exe

Installation

Dans le fichier C:\wamp\bin\php\php5.4.12\php.ini décommenter la ligne extension=php_openssl.dll
Dans le fichier httpd.conf d'apache décommenter la ligne LoadModule rewrite_module modules/mod_rewrite.so
Cloner le projet dans C:\wamp\www
Rajouter sont hostname dans l'array local de bootstrap/start.php
Mettre à jour composer en faisant "composer udpate" sur le répertoire à partir de la console
Voila !

Configuration git

git config --global push.default current
