vcpkg_install_cmake
Build and install a cmake project.
Usage:
cmake
vcpkg_install_cmake(...)
Parameters:
See vcpkg_build_cmake().
Notes:
This command transparently forwards to vcpkg_build_cmake(), adding a TARGET install
parameter.
Examples:

zlib
cpprestsdk
poco
opencv
