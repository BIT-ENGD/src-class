About
LiteStep is an add-on for World of Warcraft that adds a set of skins for Masque.
Note: You must have Masque installed to use this add-on.
Skins
This set contains the following skins:

LiteStep
LiteStep XLT

Installation
Install the package through your preferred add-on manager. You can also install the add-on manually using the following steps:

Download the package from one of the following sites:
Curse
WoW Ace
WoW Interface


Extract the package to your World of Warcraft\Interface\AddOns directory.
