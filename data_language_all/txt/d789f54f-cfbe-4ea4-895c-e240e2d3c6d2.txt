MPV's dotfiles
Usage:
On a new computer:


Configure MacOS specific settings:
shell
./fresh-mac/macos.sh


Install software:
shell
./Brewfile-install.sh


Symlink configuration files:
shell
./fresh-mac/symlink.sh


Install tools that require multiple and/or specific versions, using ASDF:
shell
./fresh-mac/asdf.py


Daily / recurring usage:


Upgrade installed tools:
shell
./Brewfile-install.sh


Add newly (manually) installed tools to version control:
shell
./Brewfile-dump.sh
