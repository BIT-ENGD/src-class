تبچی

تبچی رباتی هوشمند, بسیار ساده و البته رایگان جهت امور تبلیغاتی در تلگرام است.
نوشته شده توسط گود بوی برپایه اخرین نسخه قدیمی تلگرام.

  نصب و راه‌اندازی 🚀


ابتدا سورس تبچی‌ را کپی کرده و پیش‌زمینه‌ها را نصب کنید.
(موارد زیر را در ترمینال وارد کنید)

git clone https://github.com/turingbot/tabchi.git
cd tabchi
chmod +x bot
./bot install

 برای ساخت ربات جدید bot create/. را وارد کنید.


./bot create

حال با توجه به شناسه ای که برای ربات جدید داده شده تبچی‌ خود را راه‌اندازی کنید.
مثال:
در صورتی ک ردیس تنظیم نشده است آن را تنظیم کنید.

دستوری برای راه اندازی مجدد ردیس#
sudo service redis-server restart
دستوری برای راه اندازی ردیس#
sudo service redis-server start

./bot 1

توجه داشته باشید برای اولین بار که ربات را راه اندازی می‌کنید،از شما شناسه عددی مدیر ربات (شما و یا هرکس که می خواهید مدیر ربات شود) ، شماره ربات و کد ورود به حساب کاربری خواسته می‌شود.
 شما می توانید با استفاده از ربات @idaddybot شناسه عددی خود را بدست آورید.
از راهنمای تبچی‌ برای آشنا شدن با طرز کار رباتتان استفاده کنید.

جلوگیری از قطع شدن عملکرد تبچی‌🛡

یکی از مشکلات کار با SSH، قطع شدن آن در زمان قطع اتصال اینترنت است.وقتی اتصال اینترنت قطع می‌شود اجرای تمامی برنامه‌ها و فرامینی که در حال استفاده از SSH بودند، متوقف می‌شود. فرمان screen این‌جا به‌کمک شما می‌آید. کافی است این دستور را قبل از دستورراه‌اندازی تبچی‌ قرار دهید.
مثال:

screen ./bot 1
screen ./bot auto

برای خارج شدن از محیط screen کلید‌های ترکیبی Ctrl+A و سپس کلید D را بفشارید.برای مشاهده فهرست screen های موجود می‌توانید از دستور  screen -ls    استفاده کنید.این دستور فهرست تمامی screen های در حال اجرا را نمایش می‌دهد.برای مشاهده screen اجرا‌شده، کافی است دستور زیر را وارد کنید:

screen -r [screen name]
