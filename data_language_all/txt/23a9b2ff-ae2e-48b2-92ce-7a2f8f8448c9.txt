Installation
Add hubot-pokemon-react to your package.json file:
json
"dependencies": {
  "hubot": ">= 2.5.1",
  "hubot-scripts": ">= 2.4.2",
  "hubot-pokemon-react": ">= 0.0.0"
}
Add hubot-pokemon-react to your external-scripts.json:
json
["hubot-pokemon-react"]
Run npm install hubot-pokemon-react
Sample Interaction
user1>> ... pokemon ...
hubot>> stop it!
