flymake-go-staticcheck


Flymake backend for Go using staticcheck.
Installation
You can now install package flymake-go-staticcheck from
MELPA. Just M-x
package-installEnter flymake-go-staticcheck Enter.
Also you need install staticcheck.
shell
go get -u honnef.co/go/tools/cmd/staticcheck
Configuration
emacs-lisp
(add-hook 'go-mode-hook #'flymake-go-staticcheck-enable)
(add-hook 'go-mode-hook #'flymake-mode)
Customization
M-x customize Enter flymake-go-staticcheck
