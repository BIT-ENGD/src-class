## Typical Workflow Example
* Make LOCAL changes 
* Add any new files
    git add <file1, file2, ..., filen>
* Double-check files are modified/added/removed
    git status
* Commit changes
    git commit -a -m "Commit message"
* Push changes to remote
    git push origin <branchname>

- - - -

## Steps to Release New binary
* Make all changes
